Analyse des points d’injection avec Azure –  Enedis

## Description

Ce projet analyse les points d’injection d’électricité raccordés au réseau de Enedis, avec un focus sur la région Occitanie.

L’objectif est de :

- Comprendre l’évolution du nombre de points d’injection

- Analyser la puissance installée

- Étudier la production d’énergie injectée

- Identifier les tendances

## Définitions importantes

Point d’injection : installation raccordée qui injecte de l’électricité sur le réseau.

Puissance installée (kW) : capacité minimale et maximale de production.

## Données utilisées

Les données proviennent de l’Open Data de Enedis :

- Nombre de points d’injection

- Puissance d’injection

- Filière de production (solaire, éolien, hydraulique…)
